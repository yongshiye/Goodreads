Requesting an API Key

How to request a developer key (API key) from Goodreads.com

A developer key is also called as an API key, is the key used to access the information from the website. Most of websites require an API key to security their database from milicious calls. The API key is like a specific identity for a developer, so it will allow the website to moniter the usage of one developer. An API key is required for book.title API.
