Summary

Following these three steps...

1. Requesting an API Key

Fill out a form from this link to request a developer key from Goodreads.com.

2. Call the API

After you get your developer key, you can make your request in a HTML/JS file to call it with method GET, or just simply type in an url in the browser. you will need this base url: https://www.goodreads.com/book/title with strings indicate titles, author or other parameters for the request. The parameters accepted by this request are format (XML or JSON), key, title, author and rating.

3. Interpreting the file

After receiving the response from Goodreads.com, You can see different tags and you can choose whatever you want to display in your own Html page. Contents between two corresponding tags are the information you want from the request.
