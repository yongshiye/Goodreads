About Goodreads.com APIs

Goodreads.com is a popular social network for book lovers to exchange their thoughts. Inside this site, you can see thousands of reviews made by its members. You can get a very good sense about how worthy one book is by the rating of the book. If you are planning to start reading a new book, but can't decide which one to go for, Goodreads.com will definitely give you a great hint of that. Goodreads.com has become an Amazon company in 2013. Now it's integrated with Amazon kindle, so you can read the reviews and get the ratings more convenient for millions of books and easily pick the one you want to read most. You can write comments and share them to the community. You can, of course read reviews from other members too.

Do you know you can request those reviews by APIs too?

Requesting an API key

An developer key also called as an API key, is the key used to access the information from the website. Most of websites require an API key to protect their database from milicious calls. The API key is like a specific identity for a developer, so it will allow the website to moniter the usage of one developer. An API key is required for book.title API.
