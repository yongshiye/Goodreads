Getting Started

Following these steps to call the book.title API

1. Making the request

To make the request using book.title API, you can write a HTML/JS file to call it with method GET, or just simply type in an url in the browser. you will need this base url: https://www.goodreads.com/book/title. with strings indicate titles, author or other parameters for the request. The parameters accepted by this request are format (XML or JSON), key, title, author and rating.

Let's do an example request in detail:

1. First, you need to choose the format for your response. If you want it to be presented as XML, you should enter xml after title. (or json if you want otherwise). For example: https://www.goodreads.com/book/title.xml

2. After choosing the format, place a ? before other parameters. Then you can pass arguments into the request, for example author=Arthur+Conan+Doyle. Every word needs to be joint together by an +. If you want to add another parameter, you should put an & before it.

3. A complete sample request url should be like this : https://www.goodreads.com/book/title.xml?author=Arthur+Conan+Doyle&key=caLHbHiOFf3574vyVrdA&title=Hound+of+the+Baskervilles
(key is the developer key you get from Goodreads.com. To learn how to get a developer key, click here. You should use your own key to call the API)

Try to call it yourself! If you did so, you should get a page with a lot of information that are hard to read. How do we interpret them then?
